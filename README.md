# wine-dashboard
*Project Work 2025*

**WineMetrics** è una piattaforma web ibnterattiva progettata per il monitoraggio e l'analisi dei dati ambientali e di produzione nel settore vitivinicolo.
Questa dashboard permette agli utenti di visualizzare in tempo reale le condizioni del vigneto, inclusi temperatura, umidità, precipitazioni, ecc.

### Caratteristiche Principali


### Tecnologie Utilizzate
- ReactJS
- TailwindCSS
- Lucide-react 
- Vite

### Obiettivo
Il progetto mira a supportare viticoltori e analisti nella gestione efficiente dei vigneti, fornendo una panoramica completa dei fattori ambientali e delle performance di produzione attraverso un’interfaccia intuitiva e responsive.
